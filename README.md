# Nuxt.js frontend startup

 - `website`
    - `dist`
        - statický generovaný obsah pro umístění na staging
 - `documents`
    - zatím neexistuje, ale zde mohou být umístěné soubory mimo webovou aplikaci
 
## Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
